# Fossapp.com - MusicPlayer
Fossapp MusicPlayer is a web application that doesn't need to be installed but can be self hosted or used at https://music.fossapp.com

All you need is to deploy the server and you can use https://music.fossapp.com to play your own music.
(Recommended trying it locally with just nodeJS)
